Spaced Repetition Learning

This app was created to demonstrate a learning style called spaced repetition. I use French words to show how learning this way works. The more often the user gets a word correct, the less often the user will see it. Conversely, the less often the user gets it correct, the more often the user will see that word. This is a way to focus on the words you struggle most with.
